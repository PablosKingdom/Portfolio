Project Overview

This website portfolio was built as part of my university application in the field of computing and web development. It demonstrates my technical interests and motivation to transition into tech. The project is a personal portfolio site, featuring my CV, skills, experience, and educational background. It was made using HTML, CSS, and JavaScript and is intended for university admissions review.

Design Elements

The site includes the following front-end design elements:

  • Clean and minimalist layout with a futuristic, dark theme
  • Responsive navigation with anchor links
  • Well-structured content sections: About, Skills, Experience, Education, Contact
  • Custom CSS for visual presentation and font styling
  • Form layout for potential interaction (static only)

No back-end features were implemented, as this is a static single-page site.

UX/UI Critique: Pinterest Web Version

Pinterest offers a visually appealing and highly image-centric user interface. Its infinite scroll feature keeps users engaged by continuously loading new content without manual pagination. The masonry grid layout is both functional and visually satisfying, accommodating different image sizes seamlessly.

However, the UX suffers from aggressive prompts for user sign-up, especially for first-time visitors. Attempting to browse without an account is difficult, with most content becoming blurred or hidden after a few scrolls. This hinders casual exploration and could turn potential users away. A more subtle onboarding approach would improve retention.

Overall, Pinterest nails visual presentation but sacrifices accessibility for non-users. Improvements could include delayed sign-up prompts and limited, but open, guest browsing functionality.

Website Analytics Report

To simulate a basic marketing effort, we assume this portfolio website was promoted through an Instagram ad campaign targeting tech enthusiasts and university recruiters. The campaign led to a total of 250 visits within the first week.

  • Average time on site: 1 minute 25 seconds
  • Bounce rate: 30%
  • Top-performing section: "Projects" with 45% of all interactions
  • Traffic sources: Instagram (61%), Direct (25%), Google Search (14%)

This data suggests that Instagram was an effective platform for engagement. The "Projects" section drew the most interest, indicating it should remain a focus point. Next steps could include integrating a live contact form and more detailed project breakdowns.
